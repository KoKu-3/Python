# 🔎 개인정보 패턴 서치 (PII Pattern Scanner)

> **제작 목적**  
> 기업 내부의 비정형 데이터(텍스트 파일 등)에 방치된 민감 정보를 식별하여  
> **개인정보보호법 준수 여부를 확인**하기 위해 제작한 학습용 도구입니다.

---

## 📌 개요

기업 환경에서는 로그 파일, 설정 파일, 메모 문서 등 다양한 비정형 파일 내에 개인정보가 무심코 저장되는 경우가 있습니다.  
이 도구는 정규식(Regex) 기반의 패턴 매칭을 통해 파일 내 민감 정보를 탐지하고, 개인정보보호법 관점에서 점검이 필요한 항목을 식별합니다.

---

## 🔍 탐지 패턴 목록

| 유형 | 패턴 예시 | 관련 법령 |
|---|---|---|
| 주민등록번호 | `000000-0000000` | 개인정보보호법 제24조 |
| 휴대폰 번호 | `010-0000-0000` | 개인정보보호법 제15조 |
| 이메일 주소 | `user@example.com` | 개인정보보호법 제15조 |
| 신용카드 번호 | `0000-0000-0000-0000` | 신용정보법 제32조 |
| 계좌번호 | `000-000000-00-000` | 금융실명법 |
| 여권 번호 | `M00000000` | 개인정보보호법 제24조 |
| 운전면허 번호 | `00-00-000000-00` | 개인정보보호법 제24조 |
| IP 주소 | `000.000.000.000` | 개인정보보호법 (간접식별자) |

---

## 🚀 사용법

```bash
# 단일 파일 스캔
python pii_scanner.py -f /path/to/file.txt

# 디렉토리 전체 재귀 스캔
python pii_scanner.py -d /path/to/directory

# 특정 패턴만 선택하여 스캔
python pii_scanner.py -d ./logs --type rrn,phone,email

# 결과를 CSV로 저장
python pii_scanner.py -d ./data -o report.csv
```

## 💡 학습 포인트

- 개인정보보호법상 '개인정보'의 정의 및 유형별 분류 이해
- 정규식(Regex)을 활용한 패턴 매칭 기법 학습
- 비정형 데이터 환경에서의 개인정보 노출 리스크 인식
- 오탐(False Positive) 최소화를 위한 패턴 튜닝 방법 학습
- ISMS-P 인증 기준(개인정보 파기·보호 항목) 연계 이해
