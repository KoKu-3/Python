## 🎯 이 저장소의 목적

단순히 코드를 따라 치는 것이 아니라, **"이걸 Python으로 어떻게 만들지?"** 라는 의문에서 출발하여 직접 설계하고 구현해보는 경험을 쌓기 위해 만들었습니다.

- 관심 있는 분야의 도구를 직접 구현하며 Python 활용 능력 향상
- 라이브러리 사용법보다 **동작 원리 이해**에 집중
- 각 프로젝트마다 새로운 Python 기능이나 개념을 의도적으로 적용
